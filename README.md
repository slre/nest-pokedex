# Run on development 

1.- Clone repo

2.- Installing dependencies 
```
  npm run install OR yarn install
```
3.- Install nest CLI
```
npm i -g @nestjs/cli
```
4.- Launch Database

```
docker-compose up -d
```
## used stack
*MongoDB

*Nest
